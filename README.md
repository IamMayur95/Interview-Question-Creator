# Interview-Question-Creator



## How to run:

1. Create an environment
```bash
conda create -n interview python=3.10 -y

```


2. Activate environment
```bash
 conda activate interview
 ```

3. Intall Requirements
```bash
pip install -r requirements.txt
```
